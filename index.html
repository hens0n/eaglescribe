      Command Mode (local LLM)

      Select text → hold Ctrl+Shift+X → speak an instruction (e.g. “make this more concise”). Uses a local OpenAI-compatible server (Ollama / llama-server). Nothing is sent to the cloud.

      Example: install Ollama, run ollama pull llama3.2, keep Ollama running.

      Dictation button always toggles. Command Mode button toggles instruction capture (hotkey is hold).
